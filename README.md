# aplicación-criaturas

Aplicación CRUD desarrollada con Vue 3 y Bootstrap 5 para registrar y gestionar criaturas intergalácticas en tiempo real.

# Funcionalidades (máximo 5, más importantes)

- Registro de nuevas criaturas con formulario reactivo
- Edición y eliminación de registros en tiempo real
- Renderizado dinámico con v-for y control condicional con v-if
- Resaltado visual de criaturas no amigables
- Interfaz responsive utilizando Bootstrap 5

# Uso (si requiere o no instalar dependencias)

Requiere instalar dependencias con npm install y ejecutar el proyecto con npm run dev.
